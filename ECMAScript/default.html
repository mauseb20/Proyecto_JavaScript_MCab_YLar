Default, Rest y Named Parameters

Default Parameters

Descripción

Los parámetros por defecto de una función permiten que los parámetros formales de la función sean inicializados con valores por defecto si no se pasan valores o los valores pasados son undefined.

Caracteristicas

Si en el siguiente ejemplo no se proveyera un valor para b en la llamada, su valor sería undefined cuando se evalúe a*b y la llamada a multiplicar hubiera retornado NaN.


    ffunction multiplicar(a, b) {
      b = typeof b !== 'undefined' ?  b : 1;

      return a*b;
    }

    multiplicar(5); // 5

Con parámetros por defecto, la verificación en el cuerpo de la función ya no es necesaria. Ahora, puedes colocar simplemente 1 como el valor por defecto de b en la cabecera de la función:


    function multiplicar(a, b = 1) {
      return a*b;
    }

    multiplicar(5); // 5

Rest Parameters

Descripción

La sintaxis de los parámetros rest nos permiten representar un número indefinido de argumentos como un arreglo.

Caracteristicas

Los parámetros rest han sido agregados para reducir el codigo repetitivo que se usaba en los parámetros.


    // Antes de los parámetros rest se realizaba:
    function f(a, b){
      var args = Array.prototype.slice.call(arguments, f.length);

      // …
    }

    // Ahora es equivalente a 

    function f(a, b, ...args) {

    }

Named Parameters

Descripción

Uno de los patrones en ES6 para manejar los parámetros nombrados fue usar el patrón de objetos de opciones, adoptado desde jQuery.


    function initializeCanvas(options) {
        var height = options.height || 600;
        var width  = options.width  || 400;
        var lineStroke = options.lineStroke || 'black';
    }

Caracteristicas

Podemos lograr la misma funcionalidad usando la desestructuración como un parámetro formal para una función:


    function initializeCanvas(
        { height=600, width=400, lineStroke='black'}) {
            // Use variables height, width, lineStroke here
        }

Si queremos que todo el valor sea opcional, podemos hacerlo desestructurando un objeto vacío:


    function initializeCanvas(
        { height=600, width=400, lineStroke='black'} = {}) {
            // ...
        }
